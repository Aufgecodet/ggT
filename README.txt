Sehr geehrter Nutzer,

um dieses Pogramm zu nutzen müssen sie auf Ihrem Rechner PHP 8.1 installiert haben.

( https://windows.php.net/download#php-8.1 )

Nachdem Sie PHP heruntergeladen haben gehen Sie auf "Systemumgebungsvariablen" und erstellen eine neue Systemvariable und geben sie den pfad zu dem Ordner an, wo Sie PHP installiert haben.


Nun Navigieren Sie in dem hiesigen Ordner und öffnen die config.json datei mit einem Texteditor, dort können Sie die Einstellungen vornehmen mit welchen Zahlen das ggT berechnet werden soll.

Nun navigieren Sie mit der Commandozeile in den Ordner und geben Sie den Befehl "php ggt.php" ein.

Es sollte nun wenn sie alles richtig gemacht haben der ggT der Zahlen erscheinen die Sie in der config.json angegeben haben.
